# 迷宫生成与求解系统（数据结构课设）

## 一、项目简介

本项目是数据结构课程设计，实现了一个基于图与随机算法的迷宫生成与求解系统。  
程序能够自动生成一个合法迷宫，并保证迷宫中任意两个可达点之间存在路径，同时支持对迷宫进行路径搜索与展示。

本项目主要用于加深对 **图结构、栈/队列、随机算法以及路径搜索算法** 的理解与应用。

---

## 二、迷宫模型设计

### 1. 迷宫表示方式

迷宫采用 **二维网格（Grid）** 进行表示：

- 每个单元格代表一个迷宫节点
- 墙体用于隔开相邻节点
- 使用 `vector` 结合结构体（或 `pair<int, int>`）来表示墙的位置与节点坐标

迷宫整体可抽象为一个 **无向图**：
- 节点：迷宫中的格子
- 边：被打通的墙（可通行路径）

---

## 三、迷宫生成算法设计

### 1. 算法思想

迷宫生成采用 **随机化的连通算法**，核心目标是：

- 保证迷宫 **全连通**
- 不出现孤立区域
- 迷宫具有一定随机性

基本思路为：

1. 初始化迷宫，所有墙体均存在
2. 随机选择墙体
3. 判断打通该墙是否会破坏迷宫结构（如产生非法连接）
4. 若合法，则打通墙体
5. 重复上述过程，直到迷宫整体连通

---

### 2. 墙体处理逻辑

程序中通过遍历迷宫内部节点，筛选**可被打通的墙**：

- 仅考虑非边界墙体
- 确保墙体两侧均为合法节点
- 使用 `vector<pair<int, int>>` 存储候选墙集合

随机选择墙体进行处理，使生成的迷宫结构具有随机性。

---

## 四、迷宫求解算法

迷宫求解过程基于图的遍历算法：

- 使用 **深度优先搜索（DFS）** 或 **广度优先搜索（BFS）**
- 从起点出发，按可通行路径搜索终点
- 通过标记访问节点，防止重复访问

求解过程中可记录路径，用于最终路径的输出与展示。

---

## 五、使用的数据结构

- `vector`：存储迷宫节点、墙体列表
- `pair<int, int>` / 结构体：表示节点坐标
- 栈 / 队列：用于 DFS / BFS 路径搜索
- 二维数组：记录迷宫状态与访问标记

---

## 六、程序功能总结

- 自动生成随机迷宫
- 保证迷宫连通性
- 支持迷宫路径搜索
- 结构清晰，便于扩展与调试
